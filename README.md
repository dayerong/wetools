#### **wetools**

``` 
基于Python 3.8 & FastAPI
```

* 通过企业微信做一些自动化的运维工作，比如发布服务、日常变更等很多你能想到的功能
* 企业微信授权免登陆
* 基于LDAP及Google Authenticator认证
* gitlab的webhook自定义
